# SF FLUTTER CODING CHALLENGE

_In this challenge you will work with Polygon's Rest APIs to develop a Crypto Wallet._
<br>_Let's assume that our user (you!), has positions opened on many cryptos: One BTC, some ETH, A bunch of DOGEs... You get the point._
<br>_Your Goal is to develop a **single page APP** that contains a **Crypto Wallet** (You will find a mock a the end of this page), showing informations about those positions: Prices; charts; variations; feel free to enrich the page as much as you want. One thing is required: **Your UI must be scalable for many positions**_
